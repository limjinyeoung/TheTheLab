정렬 알고리즘 : 사용자가 입력한 데이터를 원하는 형태로 정렬하는 알고리즘
-핵심 : in-place(추가적인 메모리의 소모가 없는가) & stable(모든 알고리즘이 만족함. 같은 값에 대한 처리가 같은지)

1. 선택 정렬 n^2 : 입력된 순서대로 (i = 0; i < n-1; i++)와 (j = i+1; j < n; j++)크기 비교후 제일 작은 s를 찾아 i와 swap

2. 삽입 정렬 n^2 : 하나의 원소를 선택한 후 앞의 숫자와 비교(while)하고 앞의 숫자와 바꿈	
			**이미 정렬된 배열이나 적은 데이터에 대해서 성능이 좋음.while 루프를 탈 필요가 없기 때문.

3. 버블 정렬 n^2

4. 퀵 정렬 n(log n) - (in-place X) : pivot을 중심으로 분할하여 나누어진 배열을 재귀적으로 sort
			** 데이터가 많을때는 성능이 좋지만 데이터가 적으면 큰 효과가 없다.
			** pivot의 값이 최대값이나 최소값이 선택될경우 n^2
			** pivot을 세 값의 중위로 선정한다.

5. 머지 정렬 n(log n)

6. 힙 정렬 n(log n) - (in-place O)
